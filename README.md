## CPSC-322 Water Pollution Classification Project

This repository contains our final project for CPSC 322, where we implemented a full data-classification pipeline to predict water contamination and risk levels using machine-learning techniques.

We used a Kaggle dataset of water-quality measurements, engineered contamination and risk labels using simplified WHO/EPA guidelines, and evaluated multiple custom classifiers implemented in the mysklearn package.
